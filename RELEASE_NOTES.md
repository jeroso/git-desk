## 🩹 v0.4.1 — 충돌 머지 화면 표시 수정

v0.4.0의 3-pane 충돌 머지 화면에서 **긴 줄이 옆 칸으로 삐져나와 글자가 겹쳐 보이던 문제**를 수정했습니다.

### 🔧 수정
- 3-pane(내 것 · 결과 · 그쪽)에서 각 칸의 긴 줄이 **칸 안에서 줄바꿈**되도록 변경 — 더 이상 옆 칸 글자와 겹치지 않습니다. (들여쓰기는 그대로 유지)

### 🧩 (v0.4.0 기능 그대로)
- 충돌 파일 클릭 → 3-pane 머지, hunk별 [◀ 내 것][둘 다][그쪽 것 ▶] + 직접 편집 → 저장 시 해결 표시
- 충돌 중이면 상단 **"충돌 해결 중" 배너**(창 닫거나 앱 재실행해도 유지) → [해결하기] / [중단(abort)]
