## 🔀 v0.4.0 — 충돌 해결 도구 (3-pane 머지 + 충돌 상태 표시)

이번 버전은 머지 / 리베이스 / 체리픽 등에서 발생하는 **충돌 해결**을 크게 개선했습니다.

### 🧩 3-pane 충돌 머지 화면
- 충돌 파일을 **클릭**하면 **내 것 · 결과 · 그쪽** 3분할 화면이 열립니다.
- 충돌 구간마다 **[◀ 내 것] [둘 다] [그쪽 것 ▶]** 버튼으로 골라 넣을 수 있고, 가운데 결과는 **직접 수정**도 됩니다.
- 모든 충돌을 해결하면 **저장**으로 파일이 해결 표시(`git add`)됩니다.

### 🚧 충돌 상태 표시 + 안전장치
- 충돌이 진행 중이면 상단에 **"충돌 해결 중" 배너**가 항상 표시됩니다 — 해결 창을 닫거나 **앱을 다시 켜도 유지**됩니다.
- 배너에서 **[해결하기]** 로 충돌 화면을 다시 열거나, **[중단(abort)]** 으로 작업을 롤백할 수 있습니다.
- 이전에는 해결 창을 닫으면 충돌 중인지 알 수 없어 다른 브랜치 체크아웃 등이 막혔는데, 그 문제를 해결했습니다.

### ℹ️ 알아두기
- 3-pane은 **텍스트 충돌**(같은 파일을 양쪽이 수정)에 동작합니다. 바이너리/삭제 충돌 등은 배너의 "해결하기" 또는 "에디터에서 열기"로 처리하세요.
- 문법 하이라이트·단어 단위 비교 등 고급 표시는 아직 없습니다(추후 개선 예정).
